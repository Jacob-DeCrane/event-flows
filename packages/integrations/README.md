# @eventflows/integrations

Production-ready integrations for EventFlows (AWS, PostgreSQL, and more).

## Installation

```bash
npm install @eventflows/integrations @eventflows/core
```

## What's Included

- **InMemoryEventBus** - In-memory event bus for testing and development
- **AWS Integrations** - Coming soon (EventBridge, DynamoDB, SNS/SQS)
- **PostgreSQL Integrations** - Coming soon (Event Store, Projections, Outbox)
